
Ctrl + N - open class
Ctrl + Shift + N - open file
Ctrl + Shift + F - search by project
Ctrl + Alt + L = formatting
Ctrl + E - latest used files (+search by name)
Ctrl + Shift + E - latest changed files
Ctrl + Alt + O - optimize import

Ctrl + / - one line comment
Ctr + Shift + / - multi line comment

Ctrl + LMB - navigation to selected class/method
Ctrl + B - navigation to declaration of selected class/method
Ctrl + Alt + Left/Right - navigation by classes
Ctrl + Q - info/help for selected object
Shift + F6 - renaming at all places of usage
Alt + F1 - popup for quick selection of element for edition
Alt + Insert - code insertion (getter, setter, other methods)

Ctrl + U - navigate to superclass

Ctrl + Shift + [1-9] - put bookmark with number
Ctrl + F11 - put bookmark with selection dialog (number/character)
Ctrl + [1-9] - navigate to bookmark with this number
Shift + F11 - show bookmark list

Ctrl + Alt + T - surround selected block
Ctrl + Alt + F7 - search usages of a selected element

Alt + 1 - on/off Project view
Alt + 2 - on/off Favorites
Alt + 3 - on/off Find
Alt + 7 - on/off Structure view

Ctrl+~ switch 1.color scheme 2.keymap 3.style 4.look & feel

Alt + Shift + '+'/'-' - zoom font scale to +/-
Alt + Shift + 0 - font scale zoom to 100%

Alt + J - Select next occurrence (to edit them later simultaneously)
Alt + J - выделение по коду ниже выделенного фрагмента кода
Ctrl + P - показывает, какой параметр вы должны поставить на позицию

Ctrl + пробел - выдаст мне все доступные опции
Ctrl + Shift + пробел - ограничит варианты до подходящих по контексту
Ctrl + Shift + дважды пробел - chain completion (предложит больше подходящих вариантов (?))

Ctrl+Shift+Enter после например if или while - допишет скобки и переставит курсор между ними куда надо

Когда вы выбираете новую опцию, нажмите Tab вместо Enter! (заменяет имеющийся код на новый)

Ctrl + Alt + T - Surround with... (завернуть оператор в готовый кусок кода)

postfix expansion:
- добавить .for в конце строки, нажимаем Enter и она развернётся целиком
- ставим .sout в конец строки и нажимаем Enter
- пишем условие, ставим .if в конце строки, нажимаем Enter и и получаем полностью готовое выражение
- можно создать свой постфикс самостоятельно

Написанный код, нажав 2 раза Shift, можно добавить в live-templates

Ctrl + Shift + V - история буфера обмена

Рабочие сокращения при работе со стримами:
- tol (to list)
- ctl (collect to list)
- cts (collect to set)
- cj (collect joining)

Ctrl + W - выделяет синтаксический блок, окружающий курсор. Чем больше нажимаем сочетание, тем сильнее расширяется выделение
Ctrl + Shift + W - уменьшает зону выделения

Из коммента к статье:
- Ctrl + Shift + стрелки вверх/вниз - перемещение блока кода вверх/вниз с учетом областей видимости. Блок кода можно не выделять. Отступы корректируются автоматически.
- Alt + Shift + стрелки вверх/вниз - перемещение строки вверх/вниз. Строку можно не выделять.
- двойной рядом и внутри фигурных скобок например метода - выделяет тело метода
- двойной щелчок на номере строки с которой начинается метод - выделяет весь метод
- щелчок на номере строки в режиме отладки после того, как вы встали на breakpoint - переход исполнения (типа move to cursor) к строке с этим номером
- Ctrl + Shift + j - объединение строк
- Ctrl + Shift + LMB - переход к классу сущности, на которой щелкнули
- Ctrl + MHover - краткое описание сущности над которой завис курсор
- Ctrl + F12 - список методов класса. Если начать вводить текст - список начнет фильтроваться. Символы не обязательно по порядку: для saveAndFlush можно ввести saf
- Ctrl + Alt + z - быстро отменить локальные изменения (вообще это rollback)
- Ctrl + u - цикличное изменение регистра слова. Слово можно не выделять.

Tab - Select line in popup (and replace code without errors)

Если в диалоге открытия класса (по Ctrl+N) дописать после имени класса :456 - класс откроется на строке 456
